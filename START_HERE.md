## ❌ ISSUES?

### "Port already in use"
```bash
# Windows
netstat -ano | findstr :5504
taskkill /PID <number> /F

# Mac/Linux
lsof -i :5504
kill -9 <number>
```

### "npm not found"
- Install Node.js: https://nodejs.org/

### "Cannot connect to backend"
- Make sure both are running
- Check that you see both messages in terminal
